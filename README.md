# antioquia

![Antioquia theme image](src/assets/images/opengraph-pt.jpg)

esqueleto de um template multi-línguas para [eleventy](https://www.11ty.dev/).<br>
_multilingual skeleton template for [eleventy](https://www.11ty.dev/)._

## motivos / _motives_

por pura diversão.<br>
_for fun._

falando sério, queria começar a entender como os templates no eleventy funcionam, então decidi fazer um mini-template. <br/>
on a serious note, I wanted to understand how the templates for eleventy works, so I decided to create a mini-template to try on.
